<h1>MiniBlog</h1>

<p>This is a project for a mini blog application built with React, as part of an Udemy course.</p>

<h2>Technologies Used</h2>

<p>The project utilizes the following technologies:</p>

<ul>
  <li>React</li>
  <li>Firebase</li>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<h2>Prerequisites</h2>

<p>Before you begin, ensure that you meet the following requirements:</p>

<ul>
  <li>Have Node.js installed: <a href="https://nodejs.org/en/download/">Node.js Downloads</a>.</li>
</ul>

<h2>Installation</h2>

<p>Follow these steps to set up and run the project locally:</p>

<ol>
  <li>Clone the repository:</li>

  <pre><code>git clone https://github.com/BrunoSouza88/MiniBlog_React_Udemy.git</code></pre>

  <li>Navigate to the project directory:</li>

  <pre><code>cd MiniBlog_React_Udemy</code></pre>

  <li>Install the dependencies:</li>

  <pre><code>npm install</code></pre>

  <li>Start the development server:</li>

  <pre><code>npm start</code></pre>

  <li>Access the application in your browser at <code>http://localhost:3000</code>.</li>
</ol>

<h2>Usage</h2>

<p>The application provides a mini blog where users can create, read, update, and delete blog posts. Users can browse through existing posts, view the details of each post, edit or delete posts, and create new posts.</p>
